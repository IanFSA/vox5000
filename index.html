For best broadcast quality
  • Use wired headphones to prevent echo feedback
  • Select your external USB or XLR interface mic, not built-in laptop mic
  • Keep the level meter peaking around –12 dB, never hitting 0 dB
  • If you see red on the waveform, reduce input gain immediately
  • Monitor volume only affects what you hear — it does not affect the recording
  • Do the 5s mic test before every session
